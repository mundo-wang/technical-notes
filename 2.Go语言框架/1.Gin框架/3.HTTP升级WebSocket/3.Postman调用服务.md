我们可以使用`Postman`对这个接口进行测试。首先，了解如何访问`WebSocket`接口是关键。

例如，要使用`HTTP`访问该接口，请求的`URL`为：

```
http://localhost:8081/upgradews
```

要使用`WebSocket`访问接口，只需将`http`改为`ws`，如下所示：

```
ws://localhost:8081/upgradews
```

要删除本地分支，可以使用以下命令：

```sh
git branch -d <branch_name>
```

该命令会判断该分支的提交是否已合并到当前检出的分支，如果未合并，它会产生如下报错：

```sh
error: The branch 'dev_mundo' is not fully merged.
If you are sure you want to delete it, run 'git branch -D dev_mundo'.
```

此时，可以使用以下命令强制删除该分支：

```sh
git branch -D <branch_name>
```

`-D`本质上等价于`--delete --force`，会跳过合并检查，直接删除本地分支。

如果要删除远程分支，可以使用以下命令：

```sh
git push origin -d <branch_name>
```

其本质是向远程仓库推送一个“空引用”，从而删除对应的远程分支。

可以使用以下命令查看所有远程分支：

```sh
git branch -r   # 查看所有远程分支在本地的引用
git branch -a   # 查看所有本地分支以及远程分支在本地的引用
```

执行`git branch -r`命令前，如果希望确保看到的是最新的远程分支状态，通常需要先执行以下命令：

```
git fetch --prune
```

以同步并清理已经被删除的远程分支引用。

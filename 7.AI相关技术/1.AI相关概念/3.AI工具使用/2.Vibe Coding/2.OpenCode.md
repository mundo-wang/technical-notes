对于`Mac`用户，推荐直接用`Homebrew`安装，这是官方推荐的方式，升级也最方便：

```sh
brew install anomalyco/tap/opencode
```

安装完成后，验证是否成功：

```sh
opencode --version
```

命令执行结果如下所示：

<img src="image/image-20260706103617800.png" alt="image-20260706103617800" style="zoom:50%;" />

后续若想更新`opencode`版本，使用以下命令：

```sh
brew update
brew upgrade opencode
```

如果更喜欢桌面应用（目前还是`Beta`），也可以安装：

```sh
brew install --cask opencode-desktop
```

或者直接从官方下载页面下载对应版本的`.dmg`安装包：https://dev.opencode.ai/download。

接着我们登录`DeepSeek`开放平台：https://platform.deepseek.com/usage，充值后创建`API Key`并将其保存：

<img src="image/image-20260706103958933.png" alt="image-20260706103958933" style="zoom:40%;" />

> 需要注意，创建`API Key`后平台仅会展示一次，因此必须妥善保存；若后续遗失，则只能删除后重新创建。

在任意目录执行`opencode`命令后，会进入其专属终端。接着输入`/connect`，然后选择下方对应的选项：

<img src="image/image-20260706161838616.png" alt="image-20260706161838616" style="zoom:40%;" />

在展示出的搜索框中输入`DeepSeek`，然后从搜索结果中选择对应的选项：

<img src="image/image-20260706162030891.png" alt="image-20260706162030891" style="zoom:40%;" />

将刚才在`DeepSeek`开放平台创建的`API Key`粘贴到输入框中，然后按回车键提交：

<img src="image/image-20260706162140612.png" alt="image-20260706162140612" style="zoom:40%;" />

回到终端后，输入`/models`，点击下方对应的选项，然后选择需要使用的模型：

<img src="image/image-20260706162259127.png" alt="image-20260706162259127" style="zoom:40%;" />

在这里即可选择前面导入的`DeepSeek V4 Flash`和`DeepSeek V4 Pro`两个模型：

<img src="image/image-20260706162424595.png" alt="image-20260706162424595" style="zoom:40%;" />

选择好模型后，即可在终端输入`prompt`，开始访问并操作当前项目：

<img src="image/image-20260706162742871.png" alt="image-20260706162742871" style="zoom:40%;" />

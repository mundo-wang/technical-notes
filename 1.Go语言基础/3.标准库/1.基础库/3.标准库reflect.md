在`Go`语言中，变量分为两个部分`type`（类型信息）和`value`（值信息），它们的定义如下：

- 类型信息：在编译阶段就已确定的一组预定义元信息，用于描述变量的数据类型。涵盖基础类型、复合类型、指针类型、函数类型以及自定义类型等，用来帮助编译器理解和校验数据的使用方式。

- 值信息：变量在程序运行时实际存储的数据，具有动态变化的特性。它存放在内存中，可在程序执行过程中被读取或修改。

每个变量，都是由`type - value`这样的对（`pair`）组成的。

我们使用`reflect`包的`TypeOf`和`ValueOf`函数，来获取变量的类型对象和值对象，使用示例如下：

```go
var x float64 = 3.4
xType := reflect.TypeOf(x)   // float64
xValue := reflect.ValueOf(x) // 3.4
```

其中`xType`为`reflect.Type`类型，`xValue`为`reflect.Value`类型。

### 1. `reflect.Type`对象相关内容

对于`reflect.Type`类型对象，常用方法主要包括以下：

1. `Name()`：用于获取类型名称。只有通过`type`关键字定义的命名类型以及基础类型变量才具备类型名称；对于未命名类型，如数组、切片、`map`、指针等，该方法返回空字符串。
2. `Kind()`：用于获取类型的种类，即其底层类型。所有类型都具备种类信息，该方法返回一个`reflect.Kind`类型的值。通过调用`Kind()`方法获取到`reflect.Kind`变量后，还可以进一步调用`String()`方法，得到该类型对应的字符串表示形式。
3. `NumField()`与`Field()`：用于结构体类型，分别获取字段数量以及指定下标字段的详细信息，其中包括字段的标签信息。

操作`reflect.Type`类型对象的代码示例如下所示：

```go
type MyInt int64

type Person struct {
	Name string `mundo:"name"`
	Age  int    `mundo:"age"`
}

func main() {
	p := Person{"Alice", 30} // 不能为指针格式，否则其类型的Name为空
	pType := reflect.TypeOf(p)
	pName := pType.Name()          // Person
	pKind := pType.Kind().String() // struct
	for i := 0; i < pType.NumField(); i++ {
		field := pType.Field(i)
		tag := field.Tag.Get("mundo") // 得到标签值name和age
	}
    
	var i MyInt = 30
	iType := reflect.TypeOf(i)
	iName := iType.Name()          // MyInt
	iKind := iType.Kind().String() // int64
    
	m := map[string]interface{}{
		"name": "Mundo",
		"age":  20,
	}
	mType := reflect.TypeOf(m)
	mName := mType.Name()          // 空字符串
	mKind := mType.Kind().String() // map
}
```

### 2. `reflect.Value`对象相关内容

对于`reflect.Value`类型的对象，通常可以执行以下操作：

1. 通过调用`Int()`、`Float()`、`String()`等方法，获取其底层基础类型的值。
2. 访问切片或映射中的元素，获取结构体字段的值，以及调用结构体的方法。
3. 当参数以指针形式传入时，可以使用`Elem()`获取其底层值，并在满足可设置条件的情况下对其进行修改。

操作`reflect.Value`类型对象的代码示例如下所示：

```go
type Person struct {
	Name string
	Age  int
}

// 这里接收器需要为值类型，否则调用报错
func (p Person) Greet() string {
	return "Hello, " + p.Name
}

func main() {
	x := 42
	y := 3.14
	z := "hello world"
	vX := reflect.ValueOf(x).Int()    // 42
	vY := reflect.ValueOf(y).Float()  // 3.14
	vZ := reflect.ValueOf(z).String() // hello world

	person := Person{
		Name: "Alice",
		Age:  30,
	}
	vPerson := reflect.ValueOf(person)
	name := vPerson.FieldByName("Name").String() // Alice
	age := vPerson.FieldByName("Age").Int()      // 30
	greet := vPerson.MethodByName("Greet")
	result := greet.Call(nil)
	msg := result[0].String() // Hello, Alice

	pX := &x
	vPX := reflect.ValueOf(pX)
	vPX.Elem().SetInt(20)
	xNew := *pX // 20
}
```

反射是`Go`语言提供的一种机制，允许在运行时检查变量的类型和操作其值。通过反射，可以获取对象的类型信息、读取或修改结构体字段、调用方法等，因此在编写通用库或框架时非常有用。但反射会带来性能开销，并增加代码复杂度，使用时应谨慎。

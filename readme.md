## 一、笔记简介

这是一份由一名学习编程`2.5`坤年（六年）的小菜鸡整理的技术笔记，涵盖`Go`语言的语法、框架、第三方库的使用方法，以及数据库、`DevOps`、中间件和计算机基础等相关知识。笔记中所有内容均经过实际操作验证，确保准确性与实用性。

笔记始于`2023`年`7`月，也是我毕业后正式踏入职场的起点。工作中不断接触新知识，利用空闲时间加以整理，这不仅帮助我更系统地理解各类知识点与技术，也让我在需要时能迅速查阅、高效解决问题。通过定期回顾与更新，在温故知新中持续加深理解。

## 二、文件结构

笔记尽可能细化每个知识点，每个文件专注于讲解一个特定主题，目录结构清晰合理，相同类型的文件归入同一目录，便于通过目录名与文件名快速定位所需内容。这样的结构条理分明，方便后续对文件或目录进行增删、合并与拆分。

具体的文件与目录结构，可克隆仓库后在本地查看。
